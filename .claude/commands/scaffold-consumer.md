Invoke the `add-consumer` skill to scaffold a new consumer module.

The user will provide the module name as an argument. Examples:
- `/scaffold-consumer supplier`
- `/scaffold-consumer application`

After invoking the skill, the consumer code, SQL migration, tests, and CLI registration should all be generated. Walk through the user's responses to the skill's prompts.
